## 2. 반복문

### b. while 반복문< for 다되면 뒤로배치할것>

- `while 조건:`

  ​		`명령 블록`

- 항상 while 은 조건에 끝을 만들어 줘야된다! 조건이 True 면 무한 루프를 돈다.

```
num = 1
total = 0
while num <= 100:
	total += num     ## total = total +num
	num += 1		## num = num +1

print("total=", total)

## 이 내용을 활용해 num += 2일 경우도 구할 수 있음
```
